# nonebot_plugin_qwqa
_✨ 全自动大佬膜拜插件 ✨_

</div>

## 📖介绍
由于本插件是全自动的，所以并没有任何主动命令，目前已支持以下大佬 ~~（虽然没被授权吧）~~
- qwqa
- noa
- 白圣女
- 笨笨喵

## 💿 安装

<details open>
<summary>使用 nb-cli 安装</summary>
在 nonebot2 项目的根目录下打开命令行, 输入以下指令即可安装

    nb plugin install nonebot-plugin-qwqa

</details>

<details>
<summary>使用包管理器安装</summary>
在 nonebot2 项目的插件目录下, 打开命令行, 根据你使用的包管理器, 输入相应的安装命令

<details>
<summary>pip</summary>

    pip install nonebot-plugin-qwqa
</details>
<details>
<summary>pdm</summary>

    pdm add nonebot-plugin-qwqa
</details>
<details>
<summary>poetry</summary>

    poetry add nonebot-plugin-qwqa
</details>
<details>
<summary>conda</summary>

    conda install nonebot-plugin-qwqa
</details>

打开 nonebot2 项目根目录下的 `pyproject.toml` 文件, 在 `[tool.nonebot]` 部分追加写入

    plugins = ["nonebot_plugin_qwqa"]

</details>
